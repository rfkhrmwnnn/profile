# 🌐 Rifki Hermawan — Portfolio Website

Portofolio pribadi dengan desain **premium modern** dan animasi **cinematic**.  
Dibangun menggunakan **Next.js 14**, **Tailwind CSS**, dan **Framer Motion**.

## ✨ Fitur Utama
- Hero Section interaktif dengan parallax & typing text  
- Floating 3D elements + cursor spotlight glow  
- Mode gelap otomatis  
- Page transitions smooth (fade + slide)  
- Loading intro logo animasi  
- Testimoni, skill, & sertifikat  
- Halaman proyek detail  
- Responsif di semua perangkat
